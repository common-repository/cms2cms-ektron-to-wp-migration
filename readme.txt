Plugin for automated data migration from Ektron to WordPress. Convert Ektron to WordPress easily and fast.

== Description ==

If you wish to make the first step on the way towards your current Ektron web project improvement – CMS2CMS: Migration Plugin can assist you. It’s just a matter of a few mouse clicks to move your Ektron to WordPress without any extra skills or installations required.

= HOW THIS PLUGIN HELPS: =
- Free Demo Migration to see the transfer results first (CMS2CMS moves limited number of pages to WordPress);
- Opportunity to choose where you want to move the data from Ektron to WordPress (Data “Mapping” option);
- Migration time ranges from several minutes to several hours and depends on content volume you want to migrate to the New website;
- The Complete data migration from Ektron to WordPress depends on the amount of content you want to move to your new WordPress website.

= WHAT WILL BE MOVED FROM EKTRON TO WORDPRESS? =
* Pages along with the content included there;
* Attachments (images and videos);
* Images included in the content body.


= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Migration of SEO friendly URLs;
* Migration of keywords, metadata/meta description;
* Automated 301 redirects from existing Ektron site to the new WordPress website.


= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
- Our support team is available live 24/7.
- For more customized migration type, CMS2CMS offers data migration service packages to order.


== Installation ==
1.Download the plugin zip file.
2.Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3.Choose the downloaded file in zip format and click the “Install Now” button.
4.Go to Admin -> Plugins, activate the “CMS2CMS Ektron to WordPress migration”.
5.Register an account or sign in to the existing one.
6.Provide your Existing website URL address.
7.Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
